# Student Form

A simple **Student Data Management Form** built using HTML, CSS, and JavaScript.  
This project demonstrates basic **add**, **display**, and **delete** operations on student data using the browser’s local state (e.g., memory or local storage).

---

##  Project Overview

This web page allows users to enter student details into a form and:

 Add new student entries  
 Display the list of students  
 Delete unwanted student records

The data is dynamically rendered on the page using DOM manipulation.
